# WEB: Diseño de Inteiores

Desarrollo de una página web para un emprendimiento de diseño de interores.

## Objetivo
Mostrar los trabajos hechos y propuestas.
Ofrecer medios de contacto con los profecionales de manera sencilla.
Cartas de presentacion de las personas involucradas en el emprendimiento.

## Actualmente
- Repaso de contenido de cursos previos.
- Creacion de bocetos y busqueda de referencias online.

## Herramientas
- VSCode
- HTML
- CSS
- SASS
- Bootstrap
